# RPG Game

MonoGame 기반 2D 턴제 RPG 게임

## 실행 방법

```bash
dotnet run
```

## 조작법

- **WASD** 또는 **화살표 키**: 플레이어 이동
- **ESC**: 게임 종료
- **ENTER**: 타이틀 화면에서 게임 시작

## 구현된 기능

- ✅ 씬 시스템 (타이틀, 게임)
- ✅ 플레이어 이동 및 맵 시스템
- ✅ 카메라 시스템
- ✅ 플레이어 스탯 시스템 (HP, MP, 레벨, 경험치)
- ✅ 기본 UI (체력/마나 바)
